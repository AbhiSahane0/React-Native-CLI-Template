# React Native Starter Template (TypeScript + React Navigation)

A clean and ready-to-use **React Native CLI Starter Template** with **TypeScript** and **React Navigation** pre-configured.  
Clone, install, and start building instantly.

---

## Features
- React Native CLI (not Expo)
- TypeScript configured
- React Navigation v7 setup
- Android & iOS ready
- Simple folder structure

---

## Prerequisites
Make sure your environment is set up for React Native:  
[React Native Environment Setup](https://reactnative.dev/docs/environment-setup)

You’ll need:
- Node.js  
- Watchman (macOS)  
- Xcode (for iOS)  
- Android Studio (for Android)  
- CocoaPods (for iOS dependencies)

---

## Clone the Repository

```bash (Fire up the terminal and run)

git clone https://github.com/<your-username>/React-Native-Starter-Template.git

cd React-Native-Starter-Template

Install Dependencies

# Using npm
npm install

For iOS:
cd ios

pod install
cd ..

Run the Project
Start Metro:

npm start

Run on Android:

npm run android

Run on IOS

npm run ios

Folder Structure

React-Native-Starter-Template/
├── android/
├── ios/
├── src/
│   ├── navigation/
│   ├── screens/
│   ├── components/
│   ├── assets/
│   └── utils/
├── App.tsx
└── package.json

Build Release APK
cd android && ./gradlew assembleRelease
